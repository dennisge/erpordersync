# Store Listing Draft

## Extension Name

ERP Order Sync

## Summary

Extract orders from ERP pages and sync them to a third-party API.

## Full Description

ERP Order Sync helps operations and integration teams collect order data from internal ERP pages and forward it to downstream systems through a configurable HTTP API.

Key capabilities:

- Extract order data from ERP tables in the current browser tab.
- Configure the target table selector and ERP URL matching rules.
- Normalize ERP column names through field mappings.
- Send data in batch mode or one order at a time.
- Review debug logs inside the extension when troubleshooting is needed.

Typical use cases:

- ERP to OMS synchronization
- ERP to data platform ingestion
- Manual reconciliation and migration tasks
- Internal workflow automation

## Category

Productivity

## Single Purpose Statement

This extension extracts order data from designated ERP web pages and sends it to a configured third-party API for internal business processing.
