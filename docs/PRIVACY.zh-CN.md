# 隐私政策草稿

## 概述

ERP Order Sync 主要用于企业内部业务场景。扩展会从用户可访问的 ERP 网页中读取订单数据，并将所选数据发送到由用户或组织配置的第三方 API。

## 处理的数据

扩展可能会处理以下数据：

- 订单号
- 客户名称
- 订单金额
- 订单状态
- 订单时间
- 用户配置后在 ERP 表格中可见的其他字段

## 数据用途

扩展只会将数据用于：

- 从当前 ERP 页面提取订单信息
- 按照用户定义的字段映射转换数据
- 发送到已配置的第三方 API
- 在开启调试日志时，将调试信息保存在扩展本地存储中

## 数据存储

- 扩展配置保存在浏览器扩展存储中
- 开启调试日志后，日志会保存在浏览器扩展本地存储中
- 扩展不内置外部统计分析服务

## 数据共享

数据只会发送到用户或组织明确配置的 API 地址。

## 用户控制

用户可以：

- 修改或移除 API 地址
- 关闭调试日志
- 清空本地调试日志
- 卸载扩展以清除本地存储
