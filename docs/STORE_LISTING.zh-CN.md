# 商店介绍草稿

## 扩展名称

ERP Order Sync

## 简短介绍

从 ERP 页面提取订单，并同步到第三方 API。

## 详细介绍

ERP Order Sync 用于帮助运营、集成和实施团队从内部 ERP 页面提取订单数据，并通过可配置的 HTTP API 转发到下游系统。

核心能力：

- 从当前标签页中的 ERP 表格提取订单数据
- 支持配置目标表格选择器和 ERP 页面匹配规则
- 支持通过字段映射统一 ERP 列名
- 支持批量同步或逐条同步
- 排障时可在扩展内部查看调试日志

典型场景：

- ERP 到 OMS 的订单同步
- ERP 到数据平台的数据采集
- 人工对账和迁移辅助
- 内部流程自动化

## 分类建议

效率

## 单一用途说明

该扩展的唯一用途是从指定 ERP 网页中提取订单数据，并将其发送到已配置的第三方 API，用于企业内部业务处理。

## 建议截图

- 带“抓取订单 / 抓取并同步”按钮的弹窗
- API 与字段映射配置页
- 独立日志查看页
- 展示订单表格的 ERP 页面
